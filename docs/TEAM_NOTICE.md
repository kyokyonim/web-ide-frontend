# 팀 공유용 — EFIDE Studio UI 시안

## 팀원이 할 일

1. 아래 배포 URL(또는 로컬 실행)로 접속
2. **3가지 디자인 시안** 각각 둘러보기
3. 마음에 드는 스타일·수정할 화면을 **댓글/채팅으로 피드백**

> 로그인·버튼 등은 **목업 UI**입니다. 실제 동작/API는 없습니다.

## 시안 바로가기

| 스타일 | URL |
|--------|-----|
| Minimal (개발자 도구) | `/design/minimal/login` |
| Modern SaaS (대시보드) | `/design/saas/login` |
| Dark IDE (다크 모드) | `/design/dark/login` |

화면 **하단 바**에서 스타일·페이지(프로젝트, IDE, 관리자 등)를 바꿀 수 있습니다.

## 로컬 실행 (선택)

```bash
git clone https://github.com/kyokyonim/web-ide-frontend.git
cd web-ide-frontend
npm install
npm run dev
```

→ http://localhost:5173
